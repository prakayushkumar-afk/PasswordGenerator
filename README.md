# 🔐 Password Generator

A simple Python program that generates a random password based on user-defined length.

---

## 🚀 Features

* Generate random passwords
* User-defined password length
* Uses letters, numbers, and symbols

---

## 🛠️ Tech Stack

* Python
* Libraries used:

  * `random`
  * `string`

---

## ▶️ How to Run

1. Make sure Python is installed
2. Run the program:

```
python password_generator.py
```

---

## 📸 Sample Output

```
=== PASSWORD GENERATOR ===
Enter password length: 8
Generated Password: A7#kP2@q
```

---

## 📌 How It Works

* The program asks the user for password length
* It combines letters, digits, and symbols
* Random characters are selected to generate a secure password
